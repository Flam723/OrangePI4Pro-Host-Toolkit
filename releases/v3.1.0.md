# OrangePI4Pro Host Toolkit v3.1.0

First stable public release.

## Validated configurations

- NVMe root with boot files under `/boot`
- microSD boot + NVMe root with boot files under `/boot/boot`

## Highlights

- CAN-enabled `5.15.147-sun60iw2-can` kernel builder
- Automatic boot-directory detection
- Kernel backup and rollback
- Delayed CAN startup at 1 Mbps
- Tolerant behavior with CAN hardware disconnected
- Status, Doctor, Verify, Backup, and Report commands
- Successful CAN UUID queries on two physical systems

## Install

```bash
tar -xzf OrangePI4Pro-v3.1.0.tar.gz
cd OrangePI4Pro-v3.1.0
sudo ./install.sh
```
